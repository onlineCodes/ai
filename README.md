### how it works
Demo: [materialShower](https://itbsstefan.github.io/materialShower)
use the material Shower Theme ( duplicate n.html for a new HTML file slides-presentation ) or get setup ''npm install @shower/cli'' run shower create in local Terminal.

#### other npm commands are (after create is done)
  * ''npm run pdf'' like the Demofile [itbsstefan.pdf](https://github.com/onlineCodes/ai/blob/main/itbsstefan.pdf)
  * ''npm run archive'' makes a zip archiv
  * ''npm run bundle'' creates the folder slides or shower for ofline use
  * ''npm run serve'' run that presaentation with nodejs local after clone & ''npm install''
  * ''npm run pages'' publish in this repro commited changes on [GitHub-Pages](https://onlinecodes.github.io/ai)
